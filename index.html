About this page and the author

I am Aru, a software engineering major in SUNY Oswego. This page presents documents pertaining to my work in the Programming Languages course. Some of the documents represent work on programming assignments, code and demos. Some of the documents represent work on problem sets, including language description exercises and essays pertaining to programming languages. The work is presented chronologically with respect to due dates, and is clearly identified by means of metatext and hypertext.
